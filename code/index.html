Visualisation of GitHub repositories using d3.js

This article aims to study the most popular GitHub repositories over the last five years. The entire GitHub archive is available as a public dataset on Google BigQuery. (https://www.githubarchive.org/) The data for this work was obtained using SQL queries on this dataset. The dataset contains details such as repository name, author etc and also provides more than 20 event types including CreateEvent, CommitCommentEvent, FollowEvent, PushEvent, IssuesEvent, PullRequestEvent, etc. In order to create the plots, d3.js is used. d3 is a JavaScript library for visualising data using web standards. d3 is one of the most popular visualising techniques and uses SVG, Canvas and HTML.

The criteria to determine the most popular repository is clearly not straightforward with a number of different measures available. In this work, let us consider the number of forks as a measure of the popularity of repositories. This is certainly not the best method, but will hopefully give us some insight into the most popular repositories. The data considered in this work spans over the last five years.

A sample SQL query to extract the 100 repositories with most number of forks during the year 2015 is as follows. 

SELECT repo.name, COUNT(*) AS num FROM  [githubarchive:year.2015]
WHERE type = 'ForkEvent'
GROUP BY repo.name
ORDER BY num DESC LIMIT 100;

Increase in GitHub usage

This section will aim to study the popularity of GitHub itself. The number of forks is calculated for all the repositories. The repositories are then arranged in the order of the most number of forks. Most popular repositories over the last 5 years

This section explores the most popular repositories over the last five years in terms of the number of forks. The following plot shows five repositories with the most number of forks. The plot for each year has been normalised relative to the repository with the maximum number of forks.

It can be seen that while few repositories feature across most of the previous 5 years, there are also a number of other repositories which are popular only for one or two years. This plot also helps us to visualise the rise and fall of different open source technologies over the recent years.

Spoon-Knife which is a repository which demonstrates how to fork a GitHub repository features in the top 5 for most of the years. It is also evident from the plot that repositories like rails and html5-boilerplate which featured in top 5 during 2011 and 2012 has since then decreased in popularity. Similarly the rise of angular.js is also very clear starting from the year 2013 and has consistently featured in top 5 since then. It is also interesting to note that a repository like 2048 (a smartphone game) featuring in the top 5 for the year 2014, when the game reached its maximum popularity.

10 Most popular repositories of 2015

Finally as an extension of the previous plot, let us look at the 20 most popular repositories for the year 2015.

It is interesting to note that the number of forks for the most popular repository of 2015 is nearly as high as the total number of forks for the 100 most popular repositories of 2011. As noted earlier, this suggests an huge increase in the usage of GitHub for developing open source technologies. While bootstrap (a CSS framework by Twitter) and angular.js (a Front end javascript framework) seems to be the most popular repositories, it is worth noting the rise of other repositories such as spark , tensorflow and react . As expected technologies related to deep learning, big-data analysis and front end javascript frameworks are the most popular repositories on GitHub for the year 2015.
